XMTP · E2E encrypted

Best for remote agents. OpenClaw sends sign requests as XMTP messages to your wallet address. End-to-end encrypted. Works across networks — your agent can be anywhere.

// In OpenClaw (openclaw_tools/walletclaw_xmtp.ts) import { Client } from "@xmtp/xmtp-js" const xmtp = await Client.create(agentSigner) // Send sign request to WalletClaw address const convo = await xmtp.conversations.newConversation( "YOUR_WALLET_ADDRESS" ) await convo.send(JSON.stringify({ type: "sign_tx", to: "0x...", value: "0.01", requestId: crypto.randomUUID() })) // Listen for approval: for await (const msg of await convo.streamMessages()) { const res = JSON.parse(msg.content) if (res.txHash) console.log("Signed!", res.txHash) }
